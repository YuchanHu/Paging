# 使用Servlet的MVC分页

MVC和Servlet以及Vue3的练手项目

## 技术栈

Java、MySQL、Vue3、Bootstrap5

## 环境

**Java版本：1.8**

数据库：MySQL5.7 | 数据库可视化工具：SQLyog Ultimate

集成开发环境：IntelliJ IDEA 2020.3.1

该项目**没有使用Maven、Gradle和任何数据库连接池**

> 编码集为UTF-8，数据库的编码集也全部为UTF-8

## Servlet

该项目使用了Servlet以及FIlter技术

## Vue3

全程使用setup语法糖
